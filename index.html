🏠 우리 가족 달력

처음 한 번만 Firebase 설정이 필요해요. 아래 순서대로 따라해 주세요.

  1. Firebase Console 접속 → 새 프로젝트 만들기
  2. 프로젝트 생성 후 Realtime Database 메뉴 클릭
  3. 데이터베이스 만들기 → 테스트 모드 선택
  4. 프로젝트 설정(⚙️) → 내 앱 → 웹 앱 추가 → 설정값 복사
  5. 아래에 데이터베이스 URL만 붙여넣기
예: https://family-cal-12345-default-rtdb.firebaseio.com
설정값은 이 기기에만 저장되며, 다른 가족도 동일하게 한 번씩 설정하면 돼요.
